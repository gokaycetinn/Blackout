Install instructions

For Windows:

1. Locate the file Horror Icon UI V1 Free Edition.zip on your computer.

2. Right-click the ZIP file.

3. Select “Extract All…” from the menu.

4. Choose a destination folder and click “Extract”.

5. A new folder containing the icons will appear in the chosen location.

⸻

For macOS:

1. Find the file in Finder.

2. Double-click Horror Icon UI V1 Free Edition.zip.

3. macOS will automatically extract the contents into a new folder with the same name.

4. Open the folder to access the icons.

⸻

For Android:

1. Open the File Manager or My Files app.

2. Navigate to Horror Icon UI V1 Free Edition.zip.

3. Tap the file and select Extract or Open with a ZIP extractor app (e.g., ZArchiver).

4. The icons will be available in the extracted folder.
